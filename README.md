## Getting started

Clone the repository

```sh
git clone git@github.com:jsstrn/talks.git
```

Install all dependencies

```sh
npm install
```

Create a new deck in the `src` folder

```sh
touch src/deck.md
```

Start the build in `watch` mode

```sh
npm start
```
